A graphics algorithm has been implemented in the CPP file "main.cpp".

The program has been done with the "graphics.h" header file, hence it can be run in Dev c++.
Parameters which are need to be included :
	-lbgi -lgdi32 -lcomdlg32 -luuid -loleaut32 -lole32

The following functions have also been implemented:
1. An "Export2SVG" that creates a svg file named "1040".
2. An "Export2HTML" function that creates a html file named "1040".
